BOM Windows Object Navigator appName & appcodeName & appversion properties in JavaScript

navigator.appName - return the name of the browser. It provides vary from browser to browser. The value of the Navigator.appName property is always "Netscape", in any browser.
The value of the Navigator.appCodeName property is always "Mozilla", in any browser
appVersion - specifies the version of the browser. It returns a string representing the version information of the browser.
