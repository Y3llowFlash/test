print() - prints contents of current window confirm(message) - displays a dialog box with a message, an OK button, and a Cancel button. Returns true if the user clicked "OK", otherwise false focus() - sets focus to a window blur() - removes focus from window setInterval() - calls a function at specified intervals (in milliseconds). Continues calling the function until clearInterval() is called, or the window is closed. clearInterval() - clears a set timer with the setInterval() method. setInterval(function,milliseconds) clearInterval(intervalId) window.moveBy(x,y) window.moveTo(x,y) window.resizeBy(width,height) window.resizeTo(width,height)
