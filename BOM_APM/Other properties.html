Other properties of navigator object

cookieEnabled - returns a Boolean value indicating whether the browser has enabled cookies.
onLine - returns a Boolean value indicating whether a browser is online or false.
geoLocation - provides a geolocation object that can be used to track the user's status.
language - used to return the language version of the browser. E.g. en-us, fr, en, de.
platform - redirects the browser to the compiled site. It returns a string representing the browser's site. E.g. win32, win16, linuxi686.
