window.open() -> opens a new browser window, or a new tab, depending on your browser settings and the parameter values. Syntax: open(), open(url), open(url,target), open(url,target,windowFeatures) Value of target attributes: _blank - opens link in a new tab or window _parent - opens link in parent frame _self - opens link in current frame _top - opens link in top-most frame frame name - opens link in named frame

window.close() -> closes the current window, or the window on which it was called.
